## Kitchen Chaos Game

Приготовьте котлету на плите и следите, чтобы она не подгорела! Положите котлету на тарелку, добавьте булочку. А дальше подготовьте ингридиенты, которые указаны в заказе.  Может кто-то просит сыр, а кто-то и салат с помидорами. 
Подайте готовое блюдо оголодавшим клиентам! Приготовьте как можно больше блюд за отведенное время.
